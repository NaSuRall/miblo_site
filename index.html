Miblo est un outil de développement de serveur Rust qui permet de générer du code à partir de fichiers YAML. Il offre une approche simple et efficace pour créer des serveurs Rust en utilisant une configuration YAML.

$ cargo install miblo

Copier

$ miblo init -n rust_api -t (template file)
$ Voulez-vous générer votre api ici ? (y/n)
Y
Votre api a été générée avec succès !

$ miblo run -n rust_api

Miblo produit un projet Rust standard à partir de templates Handlebars. Code lisible, prêt à éditer.
