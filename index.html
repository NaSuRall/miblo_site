Miblo est un outil de développement de serveur Rust qui permet de générer du code à partir de fichiers template YAML. Il offre une approche simple et efficace pour créer des API Rust facilement.

$ cargo add miblo

- 3 étapes

3 commandes. 1 API.

01

Template YAML

Récupérez le fichier template YAML qui contient tous les templates nécessaires à la construction de votre API.

config.yaml
02

Générez le projet

Miblo produit un projet Rust standard à partir de templates Handlebars. Code lisible, prêt à éditer.

$ miblo init -n mon_api -t /chemin/vers/config.yaml
03

Lancez le serveur

Migrations appliquées, build automatique, serveur Axum en écoute. Votre API tourne.

$ miblo run -n mon_api
